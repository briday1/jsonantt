jsonantt
========

.. image:: ../logo.png
   :alt: jsonantt logo
   :width: 160px
   :align: right

**jsonantt** turns a plain JSON file into a polished Gantt chart (or table, or burn chart) image in one command.
No code required — describe your project in JSON, run ``jsonantt``, get a PNG.

.. code-block:: bash

   pip install jsonantt
   jsonantt project.json chart.png
